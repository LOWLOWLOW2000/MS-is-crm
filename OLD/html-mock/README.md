# HTML モック（3カラム＋フッター統一）

サーバーなしでブラウザで開ける静的な画面モックです。CSS が当たった状態で、ページ移管（リンク）も含めて確認できます。

## 開き方

- **ファイルで開く**: 各 HTML をブラウザで直接開く（`file:///...`）
- **簡易サーバー**: `npx serve .` や `python3 -m http.server 8080` で `docs/html-mock` をルートにするとリンクが確実に動作します。

## 構成

| ファイル | 画面 |
|----------|------|
| `index.html` | トップ（人類唯一の差別化… / MG_01入室） |
| `login.html` | ログイン（ダッシュボードへ / トップへ） |
| `dashboard.html` | ダッシュボード（左ナビ＋メインのリンク一覧） |
| `inside-sales.html` | インサイドセールス（IS UI） |
| `inside-sales-refinement.html` | 未調査リスト精査 |

## レイアウト

全ページ共通:

- **ヘッダー**: 3エリア（ブランド / 中央 / 右：ログイン・ログアウト等）
- **ボディ**: 3カラム（左ナビ | メイン | 右は未使用）
- **フッター**: 共通文言

IS UI のみ、メイン内で「電話制御＋リスト」と「サブタブ＋メイン」の2カラムを表示しています。

## ページ移管

- トップ → ログイン（MG_01入室）
- ログイン → ダッシュボード / トップ
- ダッシュボード → 各ナビリンク（IS UI、未調査リスト精査、KPI、AI日報等）
- IS UI / 未調査リスト精査 → ダッシュボード、相互リンク

ボタンやタブの「押したら～」などの動きは未実装です。後から JS で追加可能です。
