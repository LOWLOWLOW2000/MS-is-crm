# 完成したところまで見る手順

## 1. サーバー起動

**推奨: WSL のターミナル**でプロジェクトルートに移動し、以下を実行してください。

```bash
cd /home/mg_ogawa/SFA_MG-01/is-crm
npm run start:dev
```

- **API** (NestJS) → `http://localhost:3001`
- **Web** (Next.js) → `http://localhost:3000`

両方起動するまで十数秒かかることがあります。

### 補足: Windows で「UNC パスはサポートされません」と出る場合

プロジェクトを **UNC パス**（`\\wsl.localhost\Ubuntu\...`）で開いていると、Windows の CMD ではカレントディレクトリが使えず、`Cannot find module 'C:\Windows\scripts\...'` のようなエラーになることがあります。

- **対処**: ターミナルで **シェルを「Ubuntu」や「WSL」に切り替えて** 上記の `cd` と `npm run start:dev` を実行してください。
- `start:dev` はプロジェクトルートを `npm_config_local_prefix` で解決するようにしてあるため、同じ WSL 内でプロジェクトを開いていれば、通常はそのままでも動作します。

## 3. 見る順の例

1. **http://localhost:3000** を開く → ログイン
2. **http://localhost:3000/dashboard** で「主要メニュー」のボタンから各画面へ
3. **リスト管理** → リストを1つ作成 or 既存リストを選択 → **「このリストで架電開始」** または **架電画面を開く** から **架電専用UI** (`/calling/[listId]`) を表示
4. **設定** で人間承認 ON/OFF を確認（developer のみ変更可）
5. **レポート** で集計・CSV を確認
6. **再架電一覧** で nextCallAt 登録済みの一覧を確認
