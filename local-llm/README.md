# ローカルLLM 置き場

ローカルで動作する LLM（大規模言語モデル）関連の設定・スクリプト・連携用の置き場です。

## 想定用途

- ローカル LLM の起動・設定（Ollama / LM Studio / その他）
- 本アプリ（web / api）からローカル LLM を呼び出す際のエンドポイント・プロンプト置き場
- 推論用スクリプトやプロンプトテンプレートの格納

## ディレクトリ構成（例）

```
local-llm/
├── README.md          # 本ファイル
├── prompts/           # プロンプトテンプレート（任意）
├── config.example.*   # 設定例（任意）
└── ...
```

必要に応じて `prompts/` や設定ファイルを追加してください。
